# dotfiles

This repository contains my core dotfiles used to create a consistent and personalized development experience across different environments.

## Recommended Setup Order

To get the best experience, it is recommended to follow this order:

1.  **Shell configuration**
    Start by setting up your preferred shell:
    *   PowerShell (pwsh)
    *   Bash
    *   Zsh

2.  **Neovim**
    Install and configure Neovim to serve as your main editor, including plugins, LSP, formatting, and linting.

3.  **Starship prompt**
    Finally, install and configure Starship to provide a clean, fast, and visually appealing shell prompt.

Following this order ensures that each layer builds cleanly on top of the previous one and keeps the setup easy to troubleshoot and extend.
